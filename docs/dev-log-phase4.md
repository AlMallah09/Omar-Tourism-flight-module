# Development Log – Phase 4

## Objective

Implement authentication, authorization, and user-specific booking access.

## Tasks Completed

* Created authentication module.
* Implemented JWT token generation.
* Added password hashing using bcrypt.
* Created login endpoint.
* Configured Swagger authorization.
* Implemented current user extraction from JWT tokens.
* Protected booking endpoints.
* Linked bookings to authenticated users.
* Created GET /bookings/me endpoint.
* Restricted booking access to booking owners only.
* Tested authentication and authorization flows.

## Result

The system now supports secure authentication and user-specific access control. Users can only access and manage their own bookings.

## Status

Completed Successfully.
