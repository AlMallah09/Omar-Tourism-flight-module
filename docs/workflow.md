# Flight Booking Workflow

## Customer Workflow

1. Customer opens website or mobile application.
2. Customer views available flights.
3. Customer searches by origin, destination, and date.
4. Customer selects a flight.
5. Customer logs in or registers.
6. Customer confirms booking.
7. System saves booking in database.
8. System decreases available seat count.
9. Customer views booking confirmation.

## Admin Workflow

1. Admin logs in.
2. Admin adds or updates flight records.
3. Admin manages prices and available seats.
4. Admin views customer bookings.
5. Admin monitors system data.
