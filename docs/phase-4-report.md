# Phase 4 Report

## Phase Title

Authentication and Authorization System

## Summary

Phase 4 focused on securing the platform through JWT authentication and user authorization. User identity is now verified before accessing protected resources.

## Deliverables

* JWT authentication
* Secure login endpoint
* Password hashing with bcrypt
* Swagger authorization integration
* Protected routes
* Current user extraction
* User-specific booking management

## Outcome

The application now enforces secure access control and ensures users can only interact with their own data and bookings.
