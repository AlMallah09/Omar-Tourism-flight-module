# User Roles

## Customer

Responsibilities:
- View flights
- Search flights
- Create bookings
- View personal bookings

## Administrator

Responsibilities:
- Add flights
- Update flights
- Delete flights
- Manage flight availability
- View bookings
- Manage system data

# Use Cases

## Customer

1. Register account
2. Login
3. Search flights
4. View flight details
5. Book flight
6. View bookings

## Administrator

1. Login
2. Add flight
3. Update flight
4. Delete flight
